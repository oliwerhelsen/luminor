# Getting Started

Welcome to Luminor! This section will help you install the framework and build your first API.

## Overview

Luminor is designed to make building domain-driven applications in PHP straightforward and enjoyable. Whether you're new to DDD or an experienced practitioner, our guides will help you get up and running quickly.

## What You'll Learn

- **[Installation](installation)** - Set up Luminor in your project using Composer or the CLI installer
- **[Quick Start](quick-start)** - Build your first CRUD API step by step

## Prerequisites

Before you begin, make sure you have:

- PHP 8.2 or higher
- Composer 2.x
- A code editor (VS Code, PHPStorm, etc.)
- Basic understanding of PHP and OOP concepts

## Recommended Learning Path

1. **Start with Installation** - Get the framework installed and create a new project
2. **Follow the Quick Start** - Build a simple API to understand the basics
3. **Explore Core Concepts** - Dive deeper into DDD patterns and CQRS
4. **Try the Tutorials** - Build complete applications with step-by-step guides

{: .note }
New to Domain-Driven Design? Check out our [Domain Layer](domain-layer) documentation for an introduction to DDD concepts.
