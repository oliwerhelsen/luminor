# Tutorials

Learn by building! These tutorials guide you through creating complete applications with Luminor.

## Available Tutorials

### [Building a Product API](tutorials/product-api)

**Difficulty:** Intermediate | **Time:** 30-45 minutes

Build a complete REST API for managing products. This tutorial covers:

- Creating entities with value objects
- Implementing CQRS with commands and queries
- Building API controllers
- Using the repository pattern
- Input validation

Perfect for developers who want to understand Luminor's DDD implementation in depth.

---

### [Building a Todo API](tutorials/todo-api)

**Difficulty:** Beginner | **Time:** 20-30 minutes

Create a simple task management API. This beginner-friendly tutorial covers:

- Basic entity creation
- Simple commands and queries
- RESTful endpoint design
- Testing your API

Ideal for developers new to Luminor or DDD concepts.

---

## Choosing a Tutorial

| Tutorial        | Best For                       | Prerequisites           |
| :-------------- | :----------------------------- | :---------------------- |
| **Product API** | Learning DDD patterns in depth | Basic PHP, OOP concepts |
| **Todo API**    | Quick introduction to Luminor  | Basic PHP               |

## Learning Path

We recommend following this path:

1. **Complete the Installation** - Make sure Luminor is set up correctly
2. **Start with Todo API** - Get familiar with the basics
3. **Build the Product API** - Learn advanced patterns like Value Objects
4. **Explore the Documentation** - Dive deeper into specific features

## What You'll Need

- Luminor installed (see [Installation](installation))
- A code editor
- Terminal access
- Optional: An HTTP client like Postman, Insomnia, or curl

{: .tip }
Each tutorial includes complete source code. If you get stuck, compare your code with the provided examples.

## Beyond the Tutorials

After completing these tutorials, you'll be ready to:

- Build your own domain models
- Implement complex business logic
- Create modular applications
- Add authentication and authorization
- Deploy to production

Check out the [Examples](https://github.com/luminor-php/luminor/tree/main/examples) folder in the repository for more complete application examples.
